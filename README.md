# TSwR

Cross Entropy Method MPC dla symulowanego samochodu.
Implementacja i wykrozystanie iCEM i CEM_MPC do śledzenia zadanych trajektorii.

# Cel
Implementacja środowiska symulacyjnego samochodu autonomicznego oraz porównanie różnych podejść do sterowania predykcyjnego w zadaniu śledzenia zadanej trajektorii. Porównanie klasycznej metody MPC i CEM_MPC.

# Cel na połowe projektu
- przygotowanie działającego modelu samochodu w symulacji,
- wizualizacja przejazdu pojazdu,
- przygotowanie podstawowego kontrolera samochodzika pozwalającego na prezjechanie przygotowanej trasy.
